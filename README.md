# 教练助手 - 健身教练课程管理系统

## 架构

```
coach-system/
├── admin/                    # 管理员网站后台（教练用）
│   └── index.html            # 双击打开即可使用
├── mp/                       # 微信小程序（学员用）
│   ├── project.config.json   # 导入微信开发者工具
│   ├── app.js/json/wxss
│   ├── pages/
│   │   ├── login/            # 学员手机号登录
│   │   ├── home/             # 查看我的课程卡片（天数/次数）
│   │   ├── checkin/          # 打卡签到
│   │   ├── pause/            # 申请请假（每月<=2次）
│   │   └── history/          # 签到记录
│   └── utils/
│       ├── date.js           # 日期工具
│       └── store.js          # 数据存储
└── cloudfunctions/           # 云函数（可选，升级云开发后使用）
    ├── adminApi/             # 管理员 API
    └── studentApi/           # 学员 API
```

## 功能

### 管理员网站 (admin/index.html)
- 学员管理：添加/编辑/删除，姓名+手机号
- 课程管理：开包月卡（天数）/次卡（次数），修改天数次数
- 签到记录：查看今日签到、全部签到、撤销签到
- 请假管理：单个请假 + 集体请假（春节/出差等场景）
- 数据导出/导入（JSON 备份）

### 学员小程序 (mp/)
- 手机号登录（首次自动注册）
- 查看自己的卡：包月剩余天数 / 次卡剩余次数
- 打卡签到（包月每周<=4次提醒）
- 申请请假（每月<=2次上限）
- 签到记录查看

## 快速开始

### 1. 管理员网站
双击打开 `admin/index.html` 即可使用
- 所有数据存储在浏览器 localStorage
- 定期点右上角「导出数据」备份

### 2. 学员小程序
1. 打开微信开发者工具
2. 导入项目 -> 选择 `mp/` 目录
3. AppID 填 `wx3c83c3c8cf4bf95e`
4. 编译运行

### 数据同步
MVP 阶段：管理员在网站上录入数据 → 导出 JSON → 
小程序通过相同数据文件初始化（或手动录入）
升级方案：开通微信云开发 → 部署云函数 → 网站和
小程序共享云数据库
